# Community-Made Resources

Below are a number of community-built tools you can use to monitor the Oasis Testnet, automate your node deployment, and more. For more information on building your own integration, check out the Oasis Core Developer Docs.

# Block Explorers & Validator Leaderboards

  • Oasisscan (by bitcat)
  • Oasis Monitor (by Everstake)
  • Oasis Hubble (by Figment Networks)
  • Oasis.Fish (by Stakefish)
  • Oasis Smartstake (by Smartstake)

# Staking Dashboard

  • Anthem (by Chorus One)

# One-touch Node Deployments

  • Ankr one-touch deploy (by Ankr)
  • Bison Trails-managed Validator (by Bison Trails)

# Developer Tools

  • Rosetta Gateway (by Oasis Protocol Foundation)
  • Oasis API Server (by Simply VC)
  • Oasis RPC Proxy (by Figment Networks)
  • Oasis Indexer (by Figment Networks)

  • Panic Monitoring Tool (by Simply VC)
